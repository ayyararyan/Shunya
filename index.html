Yukti (युक्ति) · Strategy Engine

Deploy strategy logic through a Python SDK with tick-perfect backtest parity and portfolio-level controls.

Tāraṅga (तरङ्ग) · Volatility Calibration

Arbitrage-free IV surface calibration with eSSVI + HyperIV pipelines optimized for low-latency options pricing.

REST + WebSocket APIs with clean schemas for agentic automation and external system integrations.
